### 使用方式
<br/>此為Slate + NextJS 範本，使用TailwindCSS 進行Style制定
```
    npm ci  //安裝


    npm run dev //執行
```
## SlateJS Editor Example


[線上Demo](https://slate-editor.netlify.app/)


編輯器使用`SlateJS`進行撰寫，目前已完成功能有：

## 左側功能列：

-   Heading類型限制 text style 使用
-   內文`H2`大標
-   內文`H3`小標
-   `blockquote` 引言
-   數字列表、無數字列表
-   圖片插入：URL 插入、本地圖片上傳
-   客製區塊插入

## 反白功能列(控制所選文字)：

- 粗體
- 斜體
- 底線
- `Code`
- 刪除線
- 超連結
- 反白功能列依照所屬區塊進行限制
